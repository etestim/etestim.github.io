Gmail SMTP Setup

Use an App Password — required when 2-Step Verification is enabled

1
Enable 2-Step Verification

SMTP App Passwords require 2FA. Go to myaccount.google.com/security and turn on 2-Step Verification under "How you sign in to Google".

2
Generate an App Password

Visit myaccount.google.com/apppasswords. Choose app: Mail, device: Other (name it "eTestim APP"). Click Generate and copy the 16-character password shown.

Copy-paste this password immediately at eTestim - Google will not show it again. Paste the generated password to the password field of email setup at eTestim. The password is saved only on your computer encrypted and is not shared with us. Keep the 16-char generated code secure as it gives access to your account. If you close the shown password without using it at eTestim, you can delete it and generate another.
3
Use These Settings
Under GMail - add these values in eTestim
Name SurnameYour name/surname as it appears on your gmail account
Emailyour@gmail.com
Password16-char App Password generated(copy the 16-char password that gets shown on screen by selecting it and copying it as it is shown)

Outlook / Hotmail SMTP

Works for @outlook.com, @hotmail.com, and @live.com addresses

1
Sign in to Outlook Settings

Go to Outlook POP & IMAP Settings or navigate to Settings → Mail → Sync email from your Outlook inbox.

2
Enable SMTP Authentication

In Sync email, scroll to "POP and IMAP" and ensure SMTP AUTH is enabled. This allows external clients to send mail on your behalf.

Microsoft 365 / work accounts may need an admin to enable SMTP AUTH via the Exchange Admin Center.
3
Set Up App Password (if MFA is on)

If your account uses Multi-Factor Authentication, visit account.microsoft.com/security → Advanced security options → App passwords. Create one for your SMTP client.

4
Use These SMTP Settings
SMTP Configuration
Serversmtp.office365.com
Port587
EncryptionSTARTTLS
Usernameyour@outlook.com
PasswordAccount or App Password
5
Alternative: Hotmail SMTP server

If you're using @hotmail.com, you may also try smtp-mail.outlook.com on port 587 with STARTTLS. Both point to the same infrastructure.

6
Verify & Test

Configure your app with the credentials above and send a test email. If blocked, check Microsoft 365 Admin Center for security policies, or check if "Modern Authentication" is conflicting with basic auth.

Yahoo Mail SMTP Setup

Requires generating an App Password through Yahoo Account Security

1
Access Yahoo Account Security

Visit Yahoo Account Security and log into your account. This is where you manage 2FA and app passwords.

2
Enable Two-Step Verification

Under the Two-step verification section, toggle it On and follow the setup steps. This is required before App Passwords become available.

3
Generate an App Password

Scroll to Generate app password on the same page (direct link). Enter a name (e.g., "SMTP Client"), click Generate password, and copy the result.

Yahoo shows this password only once. Store it somewhere safe before closing the dialog.
4
Allow SMTP Access

In your Yahoo inbox, go to Settings → More Settings → Mailboxes. Make sure IMAP/SMTP access is enabled under Yahoo Mail Settings.

5
Use These SMTP Settings
SMTP Configuration
Serversmtp.mail.yahoo.com
Port (TLS)587
Port (SSL)465
EncryptionSTARTTLS or SSL/TLS
Usernameyour@yahoo.com
PasswordGenerated App Password
6
Test Your Setup

Plug in the settings above to your email client or app and send a test message. If authentication fails, regenerate a new app password. Yahoo also has a troubleshooting guide for SMTP issues.
